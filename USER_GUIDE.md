## Telegram бот

### Начало работы

1. Откройте бота в Telegram
2. Нажмите `/start` или кнопку "Запустить"
3. Появится главное меню с доступными разделами

### Главное меню

| Кнопка | Описание |
|--------|----------|
| 📅 Актуальные события | Просмотр ближайших мероприятий |
| 🗓 Календарь | Просмотр событий по датам |
| 🔍 Поиск и фильтры | Поиск мероприятий по параметрам |
| 📁 Архив | Прошедшие мероприятия |
| ➕ Предложить мероприятие | Добавить своё событие на модерацию |
| ❓ Помощь | Справка по боту |

### Просмотр событий

#### Список событий
- events отображаются в виде карточек
- Каждая карточка содержит: название, даты, место, формат
- Нажмите на событие для просмотра деталей

#### Карточка события
В подробной карточке отображается:
- Название мероприятия
- Даты и время проведения
- Формат (онлайн/офлайн/смешанный)
- Тип мероприятия
- Место проведения (город, страна, площадка)
- Описание
- Ссылка на регистрацию
- Контактная информация
- Кнопка "Добавить в календарь" (.ics файл)

### Календарь

- Выбор месяца через кнопки навигации
- Просмотр событий на конкретную дату
- Индикация дней с событиями

### Фильтры и поиск

Доступные фильтры:
- **По городу** - выбор конкретного города
- **По формату** - онлайн/офлайн/смешанный
- **По типу** - выставка, форум, вебинар и т.д.
- **По дате** - выбор периода
- **Поиск по названию** - текстовый поиск

Можно комбинировать несколько фильтров одновременно.

### Подача мероприятия

Пошаговая инструкция:

1. Нажмите "➕ Предложить мероприятие"
2. Заполните форму:
   - Название (минимум 5 символов)
   - Дата начала (ДД.ММ.ГГГГ)
   - Дата окончания (ДД.ММ.ГГГГ)
   - Время начала (ЧЧ:ММ или пропустить)
   - Время окончания (ЧЧ:ММ или пропустить)
   - Формат (выбор из кнопок)
   - Тип мероприятия (выбор из кнопок)
   - Город
   - Страна
   - Место проведения (или пропустить для онлайн)
   - Описание (можно в несколько сообщений, завершите словом "готово")
   - Ссылка на регистрацию (или пропустить)
   - Ссылка на источник (или пропустить)
   - Название источника (или пропустить)
   - Контактное лицо (или пропустить)
   - Телефон (или пропустить)
   - Email (или пропустить)
   - Организатор (или пропустить)
3. Проверьте сводку данных
4. Напишите "подтвердить" для отправки
5. Заявка отправится на модерацию

### Статусы заявки

После подачи заявка получает один из статусов:

- **На модерации** - ожидает проверки администратором
- **Опубликовано** - одобрено и видно в календаре
- **Отклонено** - отклонено с указанием причины
- **Черновик** - сохранено, но не отправлено

### Команды бота

| Команда | Описание |
|---------|----------|
| `/start` | Главное меню |
| `/help` | Справка по боту |

## Веб мини-апп

### Доступ

Веб-приложение открывается:
- Через кнопку меню в боте
- По прямой ссылке (настраивается через @BotFather)

### Интерфейс

#### Главная страница
- Красочный header с названием
- Сетка меню с основными разделами
- Плавающая кнопка "+" для быстрого добавления

#### Каталог событий
- Карточки мероприятий с основной информацией
- Бейджи формата (онлайн/офлайн/смешанный)
- Анимации при наведении
- Адаптивный дизайн для мобильных

#### Детали события
- Полная информация о мероприятии
- Кнопка регистрации
- Кнопка добавления в календарь
- Контакты организатора

### Особенности

- Интеграция с темой Telegram
- Тактильная обратная связь (Haptic Feedback)
- Поддержка тёмной темы
- Полная адаптивность

## Админ-панель

### Вход

1. Откройте http://localhost:8001/admin/login (или ваш URL)
2. Введите логин и пароль
3. Нажмите "Войти"

**Данные по умолчанию:**
- Логин: `admin`
- Пароль: из файла `.env`

### Дашборд

Панель управления показывает:
- Количество заявок на модерации
- Количество опубликованных событий
- Черновики
- Архивные мероприятия
- Быстрые действия

### Управление событиями

#### Все события
- Список всех мероприятий
- Фильтрация по статусу
- Поиск по названию
- Пагинация

#### Модерация заявок

Для заявок на модерации доступны действия:

1. **Одобрить**
   - Нажмите "✅ Одобрить"
   - Optionally добавьте комментарий
   - Событие публикуется в календаре

2. **Отклонить**
   - Нажмите "❌ Отклонить"
   - Укажите причину отклонения
   - Optionally добавьте внутренний комментарий
   - Заявка отклоняется, пользователь может быть уведомлён

3. **Редактировать**
   - Нажмите "✏️ Редактировать"
   - Измените необходимые поля
   - Сохраните изменения
   - После редактирования можно одобрить

#### Создание события вручную

1. Нажмите "Добавить событие"
2. Заполните форму (аналогично пользовательской)
3. Выберите статус (черновик/опубликовано)
4. Сохраните

### Статусы событий

| Статус | Описание | Видно пользователям |
|--------|----------|---------------------|
| Черновик | Сохранено, не отправлено | Нет |
| На модерации | Ожидает проверки | Нет |
| Опубликовано | Одобрено | Да |
| Отклонено | Отклонено администратором | Нет |
| Архив | Прошедшее событие | В разделе архив |

### Логи действий

В разделе "Логи" отображаются:
- Все действия администраторов
- Дата и время действия
- Тип действия (создание, редактирование, одобрение и т.д.)
- ID сущности
- Детали действия
- IP адрес (если доступно)

### Навигация

Боковое меню содержит:
- 📊 Дашборд - главная панель
- 📋 Все события - полный список
- ⏰ На модерации - заявки на проверку
- 📜 Логи - история действий
- 🚪 Выйти - выход из системы

## Часто задаваемые вопросы

### Как долго проверяется заявка?

Обычно проверка занимает от нескольких часов до 1-2 рабочих дней.

### Можно ли редактировать опубликованное событие?

Да, через админ-панель администратор может редактировать любые события.

### Что делать если заявка отклонена?

Проверьте причину отклонения. Исправьте указанные ошибки и подайте заявку заново.

### Как добавить событие в календарь?

В карточке события нажмите кнопку "Добавить в календарь". Скачается .ics файл, который можно импортировать в Google Calendar, Apple Calendar, Outlook и другие.

### Поддерживаются ли повторяющиеся события?

В текущей версии каждое событие создаётся отдельно. Для регулярных мероприятий нужно создавать отдельные записи.
