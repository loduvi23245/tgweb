# ВЭД-календарь - Telegram бот и веб-приложение

## Структура проекта

```
ved_calendar/
├── bot/                    # Telegram бот (aiogram)
│   ├── __init__.py
│   ├── main.py            # Точка входа бота
│   ├── handlers/          # Обработчики команд
│   ├── keyboards/         # Клавиатуры
│   └── states/            # FSM состояния
├── webapp/                # Веб мини-апп (FastAPI + Jinja2)
│   ├── __init__.py
│   ├── main.py            # Точка входа веб-приложения
│   ├── routes/            # Маршруты
│   └── templates/         # HTML шаблоны
├── admin/                 # Админ-панель
│   ├── __init__.py
│   ├── main.py            # Точка входа админки
│   └── routes/            # Маршруты админки
├── database/              # Работа с БД
│   ├── __init__.py
│   ├── models.py          # SQLAlchemy модели
│   ├── schemas.py         # Pydantic схемы
│   └── crud.py            # CRUD операции
├── static/                # Статические файлы
│   ├── css/               # Стили
│   ├── js/                # JavaScript
│   └── images/            # Изображения
├── templates/             # Общие шаблоны
├── config.py              # Конфигурация
├── requirements.txt       # Зависимости
├── .env.example           # Пример переменных окружения
└── README.md              # Документация
```

## Технологический стек

- **Telegram бот**: Python 3.10+, aiogram 3.x
- **Веб мини-апп**: FastAPI, Jinja2, TailwindCSS
- **База данных**: SQLite (для начала) / PostgreSQL (для продакшена)
- **ORM**: SQLAlchemy 2.x
- **Валидация**: Pydantic
- **Стили**: TailwindCSS через CDN
- **Иконки**: FontAwesome

## Установка и запуск

### 1. Установка зависимостей

```bash
pip install -r requirements.txt
```

### 2. Настройка переменных окружения

Скопируйте `.env.example` в `.env` и заполните значения:

```bash
cp .env.example .env
```

### 3. Инициализация базы данных

```bash
python -c "from database.models import init_db; init_db()"
```

### 4. Запуск бота

```bash
python bot/main.py
```

### 5. Запуск веб-приложения

```bash
python webapp/main.py
```

### 6. Запуск админ-панели

```bash
python admin/main.py
```

## Функционал

### Пользовательский бот
- Просмотр актуальных событий
- Календарный режим просмотра
- Фильтры и поиск
- Архив прошедших мероприятий
- Предложение своего мероприятия

### Админ-панель
- Управление событиями (CRUD)
- Модерация пользовательских заявок
- Фильтрация по статусам
- Логирование действий

### Веб мини-апп
- Красивый адаптивный дизайн
- Календарь мероприятий
- Карточки событий
- Фильтры и поиск
- Мобильная оптимизация
